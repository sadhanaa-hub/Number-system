Number System Program (Java)

This project is a simple Java-based application that generates multiple number series based on user input. The program takes an integer value n from the user and displays three different sequences: natural numbers, Fibonacci series, and square numbers up to n terms.

It demonstrates the use of fundamental programming concepts such as loops, variables, and user input handling using the Scanner class. The program is designed to help understand sequence generation and basic logic building in Java.

Features
Generates natural numbers from 1 to n
Displays Fibonacci series up to n terms
Calculates and prints square numbers up to n
Accepts user input dynamically

Technologies Used
Java
Scanner class

How to Run
Compile the program
Run the program
Enter the number of terms when prompted
View the generated number sequences
